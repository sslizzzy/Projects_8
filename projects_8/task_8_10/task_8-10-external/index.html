Главная страница

Добро пожаловать на мой сайт! Эта страница оформлена с помощью внешнего CSS-файла styles.css. Все стили находятся в отдельном файле и подключаются через тег <link>.

Этот абзац отмечен классом .highlight — жёлтый фон и левая полоса. Один и тот же класс можно использовать на разных страницах.

Внешний CSS — это стандарт индустрии. Один файл стилей обслуживает все страницы сайта. Если нужно изменить дизайн — достаточно поправить только styles.css.

Перейдите на страницу О проекте, чтобы убедиться, что стили работают одинаково.
