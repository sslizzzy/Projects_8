Анатомия CSS-правила

Что такое селектор?

Селектор указывает браузеру, к каким элементам применять стили. Это может быть тег (например, h1), класс (.my-class) или id (#my-id). В этом файле все правила используют селекторы по тегу.

Что такое объявления?

Объявления — это пары «свойство: значение» внутри фигурных скобок. Каждое объявление завершается точкой с запятой. В одном правиле может быть сколько угодно объявлений.

Посетите MDN Web Docs для изучения всех CSS-свойств.
